\doxysubsection{Detailed Description}
Classe base abstrata para os leitores da biblioteca. 

A classe {\ttfamily \doxylink{classLeitor}{Leitor}} representa atributos comuns para leitores e define métodos virtuais para exibir informações e calcular multas.
